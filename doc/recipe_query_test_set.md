# 菜谱查询测试集 — recipe_query_tool 覆盖度验证

> 用于测试大模型能否正确触发 `recipe_query_tool` 并返回有效菜谱知识图谱结果。
>
> 知识图谱包含 **52 道菜**，覆盖 19 种技法、21 种味道、26+ 菜系、50+ 种食材。
>
> 测试维度：**菜名精确/模糊 + 属性查询 + 反向查询 + 口语化 + 边界情况**

## 使用说明

### 测试方法

```bash
# 直接测 KG 查询（跳过模型）
PYTHONIOENCODING=utf-8 /d/anaconda3/envs/bigdog/python.exe -c "
from backend.recipe_query_adapter import query_recipe_kg
print(query_recipe_kg('小炒黄牛肉怎么做'))
"

# 测工具调用解析
PYTHONIOENCODING=utf-8 /d/anaconda3/envs/bigdog/python.exe -c "
from backend.tool_calling import _parse_textual_tool_call
print(_parse_textual_tool_call('recipe_query_tool(\"小炒黄牛肉怎么做\")'))
"
```

### 评分标准

| 维度 | 说明 |
|------|------|
| ✅ 命中工具 | 触发了 `recipe_query_tool` |
| ✅ 命中数据 | 返回了有效菜谱信息（非空/非错误） |
| ✅ 语义正确 | 返回的菜名与用户意图一致 |
| ✅ 友好兜底 | 未命中时给出建议，不崩溃 |

### 录入指引

- 正向查询（菜名+属性）：**第 1-20 条**
- 完整档案（纯菜名）：**第 21-26 条**
- 反向查询（属性→菜名）：**第 27-40 条**
- 模糊/口语化：**第 41-55 条**
- 场景化对话：**第 56-70 条**
- 边界/负面：**第 71-84 条**
- 特色数据专项：**第 85-100 条**
